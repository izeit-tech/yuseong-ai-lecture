00오늘의 흐름

ChatGPT 등장 이후 3년 5개월, 생성형 AI가 바꾼 풍경을 네 막으로 짚고 → 마지막에 내 반복업무 자동화를 직접 골라 봅니다.

막한 줄시기
① 충격ChatGPT, 인류를 뒤집다 (1억 명 2개월)2022.11~
② 폭발그림·소리·영상까지 만들다 (Sora·Suno)2023~2024
③ 진화생각하는 AI의 등장 (o1·추론)2024.9~
④ 일상AI가 직접 일을 한다 (에이전트)2025~현재
📌 핵심 메시지 이제 격차는 'AI를 쓰는 사람' vs 'AI를 안 쓰는 사람'으로 벌어집니다. 한국 직장인의 AI 업무 활용도는 미국의 약 2배(한국은행 2025.8).

② [강화] 업무가 안 떠오를 때 — AI 인터뷰

실습 1 강화 · AI 인터뷰
저는 제 반복업무가 무엇인지 잘 모르겠습니다.

제가 하루 일과를 말하면,
그중 「자주 반복되고 AI가 도울 수 있는 업무」 후보를 5개 뽑아

먼저 저에게 하루 일과를 묻는 질문 3가지를 해주세요.

03실습 2 · 오늘 실습 주제 1개 선정

좋은 후보 = 반복 · 입력 명확 · 출력 명확 · 사람 검토 가능 · 개인정보 없음. [반복업무 목록] 아래에 실습 1 결과를 붙여넣으세요.

실습 2 · 실습 주제 선정
아래 반복업무 중 오늘 실습용 자동화 주제로
가장 적합한 업무 1개를 골라 주세요.

판단 기준:
1. 60분 안 실습 가능
2. 공개자료·샘플 가능
3. 입력·처리·출력 명확
4. 표·문서로 결과 가능
5. 내일부터 바로 적용 가능

결과:
1. 추천 업무   2. 추천 이유   3. 입력 데이터
4. AI 처리 절차   5. 출력물   6. 주의할 점

04실습 3 · 공고문 요약

유성구청 → 행정정보 → 고시·공고 1건 선택 후 본문을 [공고문] 아래에 붙여넣으세요.

① 공고문 요약 (기본)

실습 3 · 공고문 요약
아래 공고문을 실무 담당자가 빠르게 이해하도록 정리해 주세요.

정리 형식:
1. 한 줄 요약   2. 핵심 요청   3. 대상   4. 기한
5. 신청·제출 항목   6. 후속 조치   7. 주의사항
8. 확인 필요 항목   9. 부서 공유용 표

부서 공유용 표 형식 : | 구분 | 내용 | 담당/확인 필요 |

주의:
- 원문에 없는 내용 추정 금지
- 날짜·기한·대상은 원문 기준
- 불명확한 내용은 「확인 필요」 표시

[공고문]

② [강화] 역할 부여

실습 3 강화 · 역할 강화

③ [강화] 부서 공유 메일로 변환

실습 3 강화 · 메일 변환
위 요약 결과를 부서 공유용 메일 초안으로 바꿔 주세요.

조건:
- 제목 1줄 + 본문 8줄 이내
- 일정·담당은 굵게 강조
- 확인 필요 항목은 맨 아래 따로
- 정중하고 간결한 공문서 어투
⚠ 실습 공통 주의 실습에는 개인정보·대외비가 없는 공개 자료만 사용하세요. AI 결과는 초안이며, 날짜·금액·대상 등은 반드시 사람이 최종 확인합니다.

05자동화 퀘스트 카드

오늘 워크숍 결과물 — 아래 5칸을 본인 업무로 채워 보세요.

  1. 줄이고 싶은 반복업무예: 매주 민원 통계 정리
  2. 입력 — 무엇을 줄까예: 민원관리시스템 엑셀 추출본
  3. 처리 — 어떻게 정리예: 동별·유형별 분류 + 처리일수
  4. 출력 — 어떤 형태로예: 부서장 보고용 1쪽 표 + 요약
  5. 사람이 확인할 부분예: 「확인 필요」 + 수치 출처
