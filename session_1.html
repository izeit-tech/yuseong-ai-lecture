00오늘의 흐름

ChatGPT 등장 이후 3년 5개월, 생성형 AI가 바꾼 풍경을 네 막으로 살펴봅니다.

  1. 충격ChatGPT, 인류를 뒤집다 (2022.11~)
  2. 폭발그림·소리·영상까지 만들다 (2023~2024)
  3. 진화생각하는 AI의 등장 (2024.9~)
  4. 일상AI가 일을 하기 시작하다 (2025~현재)
📌 오늘의 목표 AI가 어디까지 왔는지 흐름을 잡고, 마지막 실습에서 내 반복업무 중 자동화할 한 가지를 직접 골라 봅니다.

글만 쓰던 AI가 이미지·영상·음악을 모두 만들기 시작했습니다.

분야대표 도구특징

04일상 — AI가 직접 일을 한다

챗봇이 아니라 '일하는 동료'로 — AI가 마우스를 잡고 직접 실행합니다.

시점도구할 수 있는 일
2024.10Anthropic Computer Use화면을 보고 마우스·키보드를 직접 조작
2025.01OpenAI Operator가상 브라우저로 폼 작성·주문·예약 자동화
2025.07ChatGPT Agent자료 조사·문서 작성·PPT 제작까지 자동

회사 풍경이 이렇게 바뀌었다

  1. 회의 → 보고서1시간 → 5분 (자동 요약·작성)
  2. PPT 자동 제작자료만 던지면 슬라이드까지 — 디자이너 없이도
  3. 검색의 변화구글 → Perplexity·ChatGPT Search로 이동
  4. 회의록·번역회의 끝나는 즉시 요약본 + 다국어 번역
  5. 콘텐츠 자동화SNS 카드뉴스·광고 영상·BGM까지 한 번에
📌 한국 직장인, AI 활용도 미국의 2배 한 번이라도 AI를 써본 국내 근로자 63.5%, 업무 목적 사용률 한국 51.8% vs 미국 26.5%. ChatGPT 유료 가입자 수는 한국이 세계 2위입니다. (한국은행 2025.8 보고서)
⚠ 핵심 메시지 이제 격차는 'AI를 쓰는 사람' vs 'AI를 안 쓰는 사람'으로 벌어집니다. 매일 AI를 쓰는 직장인의 92%가 생산성 향상을 체감했습니다(PwC).

07자동화 퀘스트 카드 (1차)

오늘 워크숍 결과물 — 5·7·8차시에 그대로 이어 씁니다. 아래 5칸을 본인 업무로 채워 보세요.

  • 줄이고 싶은 반복업무 — 예: 매주 민원 통계 정리
  • 입력 — 무엇을 줄까 — 예: 민원관리시스템 엑셀 추출본
  • 처리 — 어떻게 정리 — 예: 동별·유형별 분류 + 처리일수
  • 출력 — 어떤 형태로 — 예: 부서장 보고용 1쪽 표 + 요약
  • 사람이 확인할 부분 — 예: 「확인 필요」 + 수치 출처
