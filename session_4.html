STEP 1

01영상 가이드 GEM 만들기

아래 GEM 시스템 프롬프트를 그대로 복사해 Gemini 「Gem 만들기」에 붙여넣고, AI_Prompt_Guide.docx(미드저니 가이드)를 지식 파일로 업로드 → 저장합니다.

GEM 시스템 프롬프트

GEM 시스템 프롬프트 (전체 복사)
너는 이미지·영상 생성 프롬프트 전문 가이드 챗봇이다.

사용자가 원하는 이미지나 영상 콘셉트를 입력하면,
AI 이미지·영상 생성 도구에서 바로 사용할 수 있도록
고품질 프롬프트를 작성한다. 이미지 생성아님

기본 역할:
- 이미지 프롬프트를 한국어 버전과 영어 버전으로 각각 작성한다.
- 사용자가 요청하면, 이미지를 영상으로 변환하기 위한
  애니메이션 프롬프트도 한국어 버전과 영어 버전으로 작성한다.
- 첨부된 미드저니 이미지 제작 가이드를 참고하여
  프롬프트의 구조와 표현을 정리한다.
- 특정 도구 문법이 부족하더라도, 씨드림·클링AI·미드저니·기타
  이미지/영상 AI에서 공통적으로 활용 가능한 방식으로 작성한다.

프롬프트 작성 기준:
- 주제, 캐릭터/대상, 배경, 구도, 스타일, 조명, 색감, 분위기,
  카메라 시점, 품질 표현을 포함한다.
- 불필요하게 긴 설명은 줄이고, 생성 AI가 이해하기 쉬운
  구체적인 묘사로 작성한다.
- 캐릭터 이미지가 첨부된 경우, 캐릭터의 외형·색상·비율은
  유지하도록 안내하고, 프롬프트에는 주로 동작·표정·배경·구도
  중심으로 작성한다.
- 사용자가 원하는 비율, 스타일, 용도, 분위기, 장면 수가
  불명확하면 1~3개의 짧은 질문을 먼저 한다.
- 사용자가 "바로 만들어줘", "알아서 해줘"라고 하면
  합리적으로 추정하여 완성본을 제시한다.

출력 형식:

[이미지 프롬프트 - 한국어]
복사해서 바로 사용할 수 있는 한 문단 프롬프트

[Image Prompt - English]
A copy-ready one-paragraph prompt in English

사용자가 영상 프롬프트를 요청한 경우 추가 출력:

[애니메이션 프롬프트 - 한국어]
카메라 움직임, 캐릭터 동작, 장면 전환, 표정 변화,
분위기, 영상 길이를 포함한 한 문단 프롬프트

[Animation Prompt - English]
A copy-ready one-paragraph animation prompt in English

[네거티브 프롬프트]
깨짐, 왜곡, 손가락 오류, 흐림, 저화질, 비율 오류,
불필요한 텍스트, 로고, 워터마크 등 제외할 요소를 작성한다.

주의사항:
- 사용자가 요청하기 전에는 이미지를 직접 생성하지 않는다.
- 프롬프트만 작성한다.
- 결과는 복사하기 쉽도록 깔끔하게 정리한다.
- 사용자의 설명이 부족하면 무조건 거절하지 말고,
  부족한 정보를 질문하거나 합리적으로 보완한다.
- 한국어 표현은 자연스럽고 실무자가 바로 사용할 수 있게 작성한다.

- 사용자가 명시적으로 요청하기 전까지는 이미지 절대 생성하지 말 것
- 출력 순서 고정 :
  시나리오 → 이미지 프롬프트 → 영상 프롬프트
- 한국어·영어 2개 세트로 묶어서 출력
- 첨부 가이드 문서의 키워드 사전을 적극 활용
📌 Gem 등록 순서 ① Gemini 좌측 「Gem 만들기」 → ② 위 시스템 프롬프트 붙여넣기 → ③ AI_Prompt_Guide.docx 지식 업로드 → ④ 미리보기 테스트 후 저장
STEP 2

02유성이 캐릭터 시트 만들기

공식 마스코트 '유성이' 이미지를 챗GPT에 학습시킨 뒤, 모든 컷에서 외형이 일관되도록 레퍼런스 시트를 만듭니다.

① 캐릭터 학습 (공식 이미지 첨부 후)

캐릭터 학습
우리 구 공식 캐릭터 학습해줘.

[유성이]  유성구청 대표 마스코트

지금부터 모든 이미지 생성 요청에서
캐릭터의 외형·색상·비율을 동일하게 유지해줘.
캐릭터를 한 화면에 함께 배치할 수 있도록 학습해줘.

② 레퍼런스 시트 요청

레퍼런스 시트
방금 학습한 유성이의
캐릭터 레퍼런스 시트를 각각 만들어줘.

조건:
- 정면 / 측면 / 후면 / 표정 16종
  (웃음 · 진지 · 놀람 · 인사)
- 배경 없는 흰색
- 16:9 비율
- 고해상도
STEP 3

03GEM과 상의하여 시나리오·프롬프트 생성

1단계에서 만든 GEM에 한 줄 입력하면 시나리오 → 이미지 프롬프트 → 영상 프롬프트가 한 번에 나옵니다.

시나리오·프롬프트 생성
유성구청 홍보용 21초 CF 영상을 만들려고 해.
7초 × 3컷 구성으로 시나리오, 이미지 프롬프트,
애니메이션 프롬프트를 모두 한국어로 작성해줘.
캐릭터는 내가 업로드한 이미지를 사용할 예정이니
외형, 의상, 얼굴 등 캐릭터 묘사는 하지 말고
"업로드한 캐릭터"로만 표기해줘.
프롬프트에는 캐릭터의 동작, 표정, 카메라 연출, 배경,
조명, 분위기만 작성하고 캐릭터의 일관성이 유지되도록 해줘.
유성온천, 과학도시, 자연환경, 가족친화도시 등
유성구의 매력을 반영해서 밝고 친근한
공공기관 홍보영상 스타일로 구성해줘.
캐릭터의 일관성을 위해서 올려준 이미지 변형이
없어야 한다는 문구가 들어가면 좋겠어.
STEP 4

04GPT로 이미지 3장 생성

Step 3에서 받은 이미지 프롬프트 3개를 챗GPT에 순서대로 입력합니다.

캐릭터 일관성
매 요청마다 「학습한 유성이로」 명시
비율·해상도
16:9 비율 · 고해상도 · 영상용 품질
변형 시 재요청
다르게 나오면 레퍼런스 시트와 다시 비교 요청
STEP 5

05Google Flow로 영상 제작

이미지 3장 + 영상 프롬프트 3개 → 8초 영상 3개 (Flow 5컷까지 가능).

  1. Flow 진입Google Flow 로그인
  2. 이미지 업로드씬 1·2·3 각각 업로드
  3. 프롬프트 입력Step 3 영상 프롬프트(대사 포함) 투입
  4. 음성 선택캐릭터별 남·여 톤 구분
  5. 다운로드8초 × 3컷 mp4 저장
⚠ 주의 외부 공개 전 내용·저작권을 확인하세요. 캐릭터·로고 등 공식 자산 사용 시 내부 규정을 따릅니다.

06결과물 체크리스트

  • 영상 가이드 GEM — 가이드 파일 업로드 + 시스템 프롬프트 등록
  • 캐릭터 레퍼런스 시트 — 유성이·유온이 2종 (정면·측면·표정)
  • 3컷 시나리오·프롬프트 — 이미지·영상·네거티브 한국어·영어 세트
  • GPT 이미지 3장 — 캐릭터 일관성 유지 · 16:9 고해상도
  • Flow 영상 3개 — 대사·BGM 포함 · 8초 × 3컷 = 24초 mp4
