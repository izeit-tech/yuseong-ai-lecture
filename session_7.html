워밍업

01기억력 장보기 게임

총 10문제 · 아이템 1초 → 4×4 배열에서 찾기. 즉석에서 타이핑 가능한 짧은 버전입니다.

기억력 장보기 게임
기억력 장보기 게임 만들어줘.

- 총 10문제 (1~5번은 1개, 6~10번은 2개 기억)
- 기억할 아이템을 1초 보여주고 사라짐
- 16개 장보기 품목(사과·바나나·우유 등)이 4×4로 나타나면
  클릭해서 정답 맞히기
- 정답·오답 표시 후 1초 뒤 자동으로 다음 문제
- 끝나면 점수 + 「다시 하기」 버튼
실습 1

025개국 음성번역기

한국어 한 번 말하면 → 5개국어로 자동 번역 + 음성 출력. (외국인 민원 응대 · 국제 행사 통역 · 다문화 가정)

5개국 음성번역기
한국어로 말하면 5개 나라말로 동시에 번역해주는 앱을 만들어줘.

기능:
- 큰 「녹음 시작」 버튼을 누르면 마이크로 한국어를 받음
- 말이 끝나면 자동으로 5개국 언어로 동시에 번역
  (영어 · 베트남어 · 프랑스어 · 일본어 · 중국어)
- 각 언어 옆에 「음성으로 듣기」 버튼 — 그 나라말로 읽어줌
- 각 결과 옆에 「복사」 버튼 — 클릭하면 텍스트가 복사됨
- 한국어 원본도 화면 위쪽에 같이 보여줌
실습 2

03QR코드 생성기

입력 → QR → 인쇄용 다운로드 (2분이면 끝). (행사 포스터 · 민원실 안내문 · 정책 홍보물)

QR코드 생성기
주소나 글자를 입력하면 QR코드를 만들어주는 앱을 만들어줘.

기능:
- 입력칸에 주소(URL)나 글자를 넣고 「QR코드 만들기」 버튼 누르기
- QR코드가 화면에 크게 표시됨
- QR 코드에 올려준 이미지가 가운데 나오도록 해줘.
- 다운로드 / 클립보드로 복사 기능 있어야함
실습 3

04뽀모도로 앱

원형 타이머 · 10분 단위 설정 · 알림 · 멈춤 · 초기화.

뽀모도로 앱
뽀모도로 앱을 만들고싶어.
원형 타이머이고 시간을 10분단위로 설정할 수 있도록 해줘.
시간을 설정하고 시작을 하면 시간이 줄어들고 0이 되면 알림을 해줘.
멈춤기능, 초기화 기능이 있으면 좋겠어.
