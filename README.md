# 유성구청 실무교육 — 생성형 AI가 바꾸는 직장 업무변화

유성구청 실무교육 강의안(HTML)입니다. 외부 CDN 없이 단일 HTML 파일로 동작합니다.

## 구성

| 파일 | 내용 |
|------|------|
| `session_1.html` | 1차시 — 생성형 AI가 바꾸는 직장 업무변화 |
| `session_2.html` | 2차시 (예정) |
| `session_3.html` | 3차시 (예정) |
| `session_4.html` | 4차시 (예정) |
| `index.html` | 강의 자료실 메인 (예정) |

## 주요 기능

- 차시별 sticky 내비게이션 / 섹션 앵커 바로가기
- 실습 프롬프트 박스 — **복사 버튼 클릭 시 클립보드 복사 + 토스트 안내**
- 노트 / 경고 / 팁 박스, 테이블, 번호 스텝, 체크리스트 컴포넌트

## 사용

각 HTML 파일을 브라우저에서 열면 됩니다. 별도 빌드·서버가 필요 없습니다.
