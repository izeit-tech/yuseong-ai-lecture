02개인 맞춤 설정 — 커스텀 인스트럭션

한 번 설정하면 모든 대화에 자동 적용되는 '내 업무 비서' 만들기.

📌 설정 위치 Gemini 설정 → 개인화 → 맞춤 안내 · 한 번 입력하면 새 대화마다 자동 적용, 언제든 수정 가능.

설정문 ① 전문가 관점 시스템 (E=n)

아래 설정문을 '맞춤 안내'에 붙여넣으면, 프롬프트 끝에 E=1 / E=2 / E=3만 붙여 전문가 관점 답변을 받을 수 있습니다.

커스텀 인스트럭션 · 전문가 관점
[역할 정의]
사용자가 요청하는 주제에 대해
최고 수준의 전문가 관점을 통합적으로 제시합니다.

[지시사항]
E=1, E=2 등 특별 요청 시 전문가 관점에서 답변
· E=1: 상위 1% 권위자 1명
· E=2: 서로 다른 관점 2명
· E=3: 다양한 시각 3명

[출력 형식]
전문가 소개 / 핵심 답변 / 종합 결론

설정문 ② 거짓말 방지 규칙

AI가 모르는 건 '모른다'고 말하게 하는 4줄 규칙 — 공무원 업무 필수. 1차시 '환각' 문제의 절반을 차단합니다.

커스텀 인스트럭션 · 거짓말 방지
아래 규칙을 엄격히 준수해서 답변해주세요.

1) 근거가 없어 불확실할 경우,
   절대 지어내지 말고 "알 수 없습니다"라고 명시해주세요.

2) 답변 전 정보를 검증하고, 출처가 불분명한 부분은
   "확실하지 않음"이라고 표시해주세요.

3) 확실한 정보만 사용하되, 추측이 불가피하면
   "추측입니다"라고 밝혀주세요.

4) 문의가 모호하면 먼저 맥락이나 세부 정보를 더 요청하세요.
⚠ 주의 공식 보고서·법령 인용 시에는 규칙을 설정했더라도 반드시 사람이 최종 검토하세요.

03활용 1 · 전문가 관점 시스템 (E=n)

설정문을 넣었다면, 이제 프롬프트 끝에 E=숫자만 붙이면 됩니다.

E=n 사용 예시

05딥리서치로 전문가 수준 자료 조사

하나의 주제를 수십 개 소스 기반으로 심층 조사 — '사고 모델' 선택 권장, Google 문서로 바로 내보내기.

딥리서치 · 대덕특구 정책 조사
「대덕연구개발특구 입주기업 지원 활성화 방안」에 대한
심층 조사를 해줘.

조사 관점:
1) 한국 연구개발특구 지원 정책의 한계점과 사각지대
2) 한국과 산업 구조가 유사한 국가의 R&D 클러스터 사례
3) 한국 적용 시 예상되는 법·제도·예산상 장애요인
4) 최근 2~3년 내 새롭게 등장한 정책 키워드
⚠ 주의 5~10분 대기 · 100개 이상 자료 종합 · 결과물은 반드시 출처 확인 후 사용.
